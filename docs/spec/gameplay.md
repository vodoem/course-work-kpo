# Правила и параметры игры

## Поле и координаты
- Игровое поле прямоугольное.
- Дрон движется по оси X в нижней части поля.
- Падающие объекты летят сверху вниз.

## Очки кристаллов
- Голубой кристалл: 8–10 очков
- Зелёный кристалл: 5–7 очков
- Красный кристалл: 1–4 очка

## Базовые скорости (px/tick)
- Дрон: 4
- Кристаллы: 2
- Астероиды: 3

Рост скорости с уровнем:
v = v_base * (1 + 0.015 * (level - 1))

## Чёрная дыра
- Радиус притяжения: 220 px
- Радиус ядра: 40 px
- Вектор ускорения (идея): a = G / (d^2 + ε) в направлении центра
- Эффекты: потеря энергии + дезориентация (временная инверсия управления)

## Бонусы
### Ускоритель
- Множитель скорости дрона: 1.6
- Множитель очков: +25%

### Стабилизатор времени
- Замедление опасных объектов: -35% к скорости падения
- +5 секунд к таймеру уровня

### Магнит
- Радиус: 120 px
- Кристаллы внутри радиуса ускоряются к дрону и «автособираются» при приближении

## Условия завершения уровня
- Победа: score >= RequiredScore и собраны все типы кристаллов (логика целей).
- Поражение: timeLeftSec <= 0 или energy <= 0.
