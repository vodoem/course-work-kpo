# Сводка соглашения по стилю C# (кратко)

Эта сводка не заменяет оригинальный документ, но фиксирует правила, которые агент должен соблюдать.

## Форматирование
- Отступы: 2 пробела, табы запрещены (заменять на пробелы).
- Скобки `{}` на отдельных строках.
- Каждая команда на отдельной строке.

## Файлы
- 1 файл = 1 тип (класс/enum/record и т.п.).

## Комментарии
- Все объявления снабжать XML-комментариями `///`.
- Внутри методов — обычные комментарии `//` по необходимости.

## Именование
- Классы/методы/события/свойства: PascalCase.
- Поля: private + `_camelCase`.
- Константы: `UPPER_SNAKE_CASE`.
- Локальные переменные: camelCase.

## Параметры методов
- обычные: префикс `par`
- ref: `refX`
- out: `outX`
